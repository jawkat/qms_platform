Chaque fichier JSON doit contenir un objet avec les clés texte, version et rows (ou articles). Le tableau rows contient les articles avec les champs obligatoires : ARTICLE, CONTENU, TYPE_OBLIGATION.

texte

code, titre, type (obligatoires), domaine, sousdomaine, theme, referentiel, secteurs

version

numero_version (obligatoire), date_publication, statut, preambule

rows

ARTICLE, CONTENU, TYPE_OBLIGATION, RISQUE, MOTS_CLES, RESUME, PREUVE_CONFORMITE

Vous pouvez sélectionner plusieurs fichiers JSON à importer en même temps.
