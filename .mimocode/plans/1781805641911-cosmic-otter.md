## Stratégie de Migration

1. **Phase 1 en premier** — Unification NC est le prérequis (touché par presque tout)
2. **Phase 2 après Phase 1** — HSE dépend des NC unifiés pour les incidents/NC
3. **Phases 3-8 indépendantes** — Peuvent être faites dans n'importe quel ordre

### Commandes migration
```bash
# Pour chaque phase, générer et appliquer:
flask db migrate -m "phase_X_description"
flask db upgrade

# Seeds initiaux:
python bootstrap_database.py
```
